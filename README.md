# Student Management System

This repository contains two versions of a Java Student Management System developed for an AI-assisted programming assessment.

## Manual version

The Manual version includes:

- Multiple student registration
- Multiple course registration
- Online and physical course types
- Basic exception handling
- Total fee calculation

## AI-assisted version

The AI-assisted version includes:

- Menu-driven system
- Multiple students and courses
- Search student by ID
- Display all students
- Stronger input validation
- Duplicate checking
- File saving and loading
- Improved class organization

## Project Structure

- `manual-version` — Manual Java code
- `ai-assisted-version` — AI-assisted Java code
- `screenshots` — Testing evidence
- `uml-diagrams` — UML diagrams

## Technologies Used

- Java
- IntelliJ IDEA
- Git
- GitHub

## How to Run

1. Open the project in IntelliJ IDEA.
2. Open the required `StudentManagementSystem.java` file.
3. Run the class containing the `main` method.
4. Enter values through the IntelliJ Run console.
